12-bit register

In onze 12-bit register wordt de 12-bit data d input gesplit. Wanneer write aanstaat en de clock op falling edge is, zal elke bit individueel in een D Flip-Flop gestoken worden.

De 12-bit output q zal de outputs van all D Flip-Flops weer achter elkaar plakken tot een 12-bit getal.

Register file

Voor deze opdracht hebben we onze 12-bit registers uit de vorige opdracht hergebruikt. Er is een rs en rt die een 3-bit input nemen. Hiermee kan er verwezen worden naar 1 van de 8 registers in de register file. Doormiddel van een multiplexor zullen de waarden van de geselecteerde inputs doorgegeven worden naar de outputs rs en rt respectivelijk.

Er is ook een 3-bit input rd die aangeeft naar welk register in de register file geschreven zal worden. Er wordt geschreven op falling-edge en wanneer write op 1 staat.

Counter

De counter maakt gebruik van een register uit de eerste opdracht en de 12-bit CLA uit opdracht 2. De adder heeft als inputs de output van het register en een 0000 0000 0001. De uitkomst is dus telkens wat er in het register zit + 1. Je kunt dit ook berijken door de tweede input op 0 te zetten en de CarryIn op 1.

De outkomst van deze adder wordt dan terug in het register gestoken die altijd zal writen op falling edge (door een constante 1).

Verder is er nog een 12-bit input Reset match. Wanneer de uitkomst groter of gelijk is aan deze waarde, zal het register gereset worden.

Finite state machine

Voor deze opdracht gebruiken we de counter uit de vorige vraag. We zullen als reset waarde 28 nemen waarna de counter terug op 0 zal springen. Dit omdat de stoplichten cycli niet verder dan 28 ticks gaan. De counter zal optellen op elke falling edge. We zullen de output van de counter telkens vergelijken met een reeks getallen.

Voor het eerste stoplicht zullen we kijken of de counter kleiner is dan 8 voor een groen licht. Kleiner dan 12 en groter of gelijk aan 8 voor een geel licht. Als de counter een andere waarde heeft, zetten we het licht op rood.

Voor het tweede stoplicht moesten we iets meer voorwaarden in rekening nemen. Maar ook voor dit stoplicht bepalen de ranges waartussen de counter waarde kunnel liggen, de kleur van het stoplicht.
